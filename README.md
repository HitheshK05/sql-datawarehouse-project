
# 🏗️ Data Warehouse & Analytics Project

## 📌 Overview

I built this project to implement a complete SQL Server-based Data Warehouse from scratch. The goal was to integrate data from multiple business systems, transform it using ETL processes, and organize it into a dimensional model optimized for analytical reporting.

This project covers the complete data engineering workflow, including data ingestion, data cleansing, dimensional modeling, ETL development, and SQL-based analytics.

---

## 🎯 Project Objectives

- Build a centralized Data Warehouse using SQL Server.
- Integrate ERP and CRM datasets.
- Perform data cleansing and validation.
- Design Fact and Dimension tables using a Star Schema.
- Develop reusable ETL processes.
- Generate business insights through SQL queries.

---

## ⚙️ Project Implementation

### Data Engineering

During this project, I implemented the following:

- Imported data from multiple source systems.
- Cleaned and validated incoming datasets.
- Designed a dimensional data model.
- Created Fact and Dimension tables.
- Developed ETL scripts for loading and transforming data.
- Optimized SQL queries for better performance.
- Documented the database design and ETL workflow.
